## Abstract Factory
##### მაგალითად გვაქვს ავეჯის მაგგაღია და გვაქვს მოდერნული მაგიდა თავის სკამებთან ერთად და ძველებური მაგიდა თავის სკამებთან ერთად. თუ გვინდა, რომ არ აგვერიოს მოდერნული და ძველი:

ცალცალკე კლასებს შევუქმნით, რომელიც ერთი დაიგივე პროტოკოლს დაემორჩილებიან და კლიენტს ექნება ამ პროტოკოლს დამორჩილებული ობიექტი. როდესაც გამოიძახებს getTable() და მერე რომ გმაოიძახოს getChairs() დარწმუნებული ვიქნებით, რომ არ აირევა, რადგან კონკრეტულ მოდერნულ ან ძელს დააბრუნებენ მაგალითად ორივე უეჭ.
